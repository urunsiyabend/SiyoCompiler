# Markdown Syntax Guide

This post demonstrates the supported Markdown features.

## Text Formatting

Use *single asterisks* for italic and **double asterisks** for bold.

Wrap text in `backticks` for inline code.

## Headings

Use # for h1, ## for h2, and ### for h3.
