# Getting Started

Siyo is a modern programming language that compiles to JVM bytecode.

## Installation

Download the latest release and add `siyoc` to your PATH.

## Your First Project

Create a new project with `siyoc new myapp` and start coding.

### Writing Code

Siyo uses **clean syntax** with *type inference* and powerful features like actors, channels, and Java interop.

Check out the [documentation](https://github.com/siyo-lang) for more details.
